Estadisticas - Examen
Falso y Verdadero

1. Estadística Es una Ciencia que explica y provee de herramientas para trabajar con datos, ha experimentado un gran desarrollo a lo largo de los últimos años..



2. Actualmente se aplica en todas las áreas del saber, por ejemplo en Sociología, Educación, Psicología, Administración, Economía, Medicina, Ciencias Políticas, entre otras.



3. La frecuencia: se define como el conjunto de unidades, el cual corresponde al numero o porcentaje en la que no se presenta una caracteristica.



4. La frecuencia relativa se presenta como el numero de veces que se repite una caracteristica.



5. La frecuencia absoluta se presenta como el porcentaje en base al numero de veces que se repite una caracteristica.



6. El Grafico de Barras se utiliza generalmente para representar la frecuencia de las categorias de una variable cuantitativa.



7. El Histograma mediante este se representa o se muestra la frecuencia de una varible cuantitativa, en la cuales cada barra representa la frecuencia de la variable en la poblacion de estudio (o de ser el caso la muestra).



8. La poblacion es un conjunto de unidades de analisis provenientes de una muestra.
